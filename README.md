# 🎉 Event Management System (Java)

A simple console-based Event Management System developed using Java.  
This project allows users to manage events with basic CRUD operations.

---

## 📌 Features

- ➕ Add Event
- 📋 View All Events
- ✏️ Update Event
- ❌ Delete Event
- 🖥 Console-based interactive menu

---

## 🛠 Technologies Used

- Java
- OOP (Object-Oriented Programming)
- ArrayList (Java Collections Framework)

---

## 📂 Project Structure

EventManagementSystem/
│── Event.java
│── EventManager.java
│── Main.java

---

## 🚀 How to Run

1. Clone the repository:
   git clone https://github.com/heragopika2006/EventManagementSystem.git

2. Navigate into the folder:
   cd EventManagementSystem

3. Compile:
   javac *.java

4. Run:
   java Main
